DAS KONFIGURATIONSPROGRAMMMit dem Konfigurationsprogramm können Sie die EURACOM 180/1/2 einfach undübersichtlich programmieren. Alle Einstellungen, die Sie normalerweise mit dem Telefon vornehmen müßten, können Sie bequem am Bildschirm vorneh-men und auf Ihrer Festplatte oder einer Diskette sichern.Dieses Dokument enthält Informationen, die Sie für die Installation des Konfigurationsprogramm EURACOM 180/1/2 benötigen. Es ist eine von mehrerenDateien, die sich auf dieser Diskette befinden.Auf dieser Diskette befinden sich folgende Files:- INSTALL.BAT- KONFIG.DOC- KONFIG.EXE- KONFIG.HLP- README.TXTZusätzlich werden Ihnen wichtige Informationen geben, die nicht in die gedruckte Dokumentation aufgenommen werden konnten. * WICHTIG - WICHTIG - WICHTIG - WICHTIG - WICHTIG - WICHTIG* Zur Vermeidung von Falschwahl wählen Sie an einem * Endgerät mit dem Mehrfrequenzwahlverfahren (MFV) vor der * ersten Benutzung eine Ziffer. Damit erkennt die EURACOM 180/1/2* das Wahlverfahren.* Bei jedem Wechseln des Wahlverfahrens müssen Sie vor der * ersten Benutzung eine Ziffer > 2 wählen.* Betreiben Sie zwei Endgeräte an einem Anschluß (z.B. mit * einer AWADo) müssen beide Endgeräte mit dem gleichen * Wahlverfahren wählen.Das Installationsprogramm installiert diese Files für Sie im gewünschten Verzeichnis auf Ihrer Festplatte.SIEHE AUCH...   Bedienungsanleitung EURACOM 180/1/2   Kapitel "Konfigurieren der EURACOM 180/1/2 vom PC"BEACHTEN SIE:Voraussetzung zur Programmierung der EURACOM 180/1/2 vom PC ist eine Verbin-dung zwischen der EURACOM 180/1/2 und Ihrem PC. Das erforderliche 1:1-Ver-bindungskabel ist NICHT Bestandteil des Lieferumfanges.INSTALLIERENGeben Sie INSTALL ein.Das Installationsprogramm fordert Sie dazu auf, das gewünschte Verzeichnis für das Konfigurationsprogramm anzugeben, und kopiert dann alle Dateien von der Disketten auf Ihre Festplatte. Nach Abschluß der Installation befinden Sie sich in dem Verzeichnis, in das die Files geschrieben wurden.NETZWERK, VIRENSCANNERBeachten Sie bitte, daß bei der Auswahl Optionen durch Änderungdes COM-Ports, der Farben oder des Paßworts Ihre KONFIG.EXE modifiziertwird.STARTENSie starten das Programm mit KONFIG. Eine ausführliche Hilfe, die Sie mit <F1> aufrufen, steht Ihnen zu jedem Menüpunkt und zur Bedienung des Programms zur Verfügung.
